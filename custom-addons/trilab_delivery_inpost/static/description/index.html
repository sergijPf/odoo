Trilab Delivery InPost

Odoo module to integrate with InPost Courier services.

  1. InPost definition on delivery method (including TEST and PROD environment)
  2. List InPost locker synchronized with inPost database/API and available on sale order
  3. Possibility of manual inPost shipping creation and label download (also as mass action)
  4. Website (frontend) integration is available in separate module (Trilab Delivery InPost Website)

Supports Odoo version 15.
